# flowpipe help
Display help and usage information for any command in the application.

## Usage
```bash
flowpipe help [command] [flags]
```

## Examples
Show help:
```bash
flowpipe help 
```
Show help for the `mod` sub-command:
```bash
flowpipe help mod
```
Show help for the `mod install` sub-command:
```bash
flowpipe help mod install
```
